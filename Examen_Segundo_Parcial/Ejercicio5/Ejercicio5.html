Ejercicio5: Haz un programa mediante un formulario HTML que solicite un numero inicial y un numero final, el cual se procesara usando el metodo GET en un script PHP. El script debe validar que el numero inicial sea menor que el final y que ambos contengan valores enteros; el programa debera mostrar todos los numeros comprendidos en ese rango que sea multiplos de 3 o multiplos de 5. En caso de que no existan multiplos en el rango dado o los numeros sea iguales, el sistema debera informar al usuario con un mensaje de advertencia claro.
